# Unity.Extensions

Extensions helper methods for Unity that I use frequently

## Installation

Add the dependency to your `manifest.json`

```json
{
  "dependencies": {
    "jd.boiv.in.extensions": "https://github.com/starburst997/Unity.Extensions.git"
  }
}
```

## Usage

```csharp
var vector3 = myFloat.ToV3();
```
